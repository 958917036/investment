## 二、结论先行原则（强化）

每个L2子章节（基本面/技术面/资金面/情绪面）分三层：

### 第1层：「核心矛盾」引用块（markdown `>` 引用）
- 一句话结论，高度凝练
- 包含方向判断（看多/看空/中性）和核心逻辑
- 读者扫过引用块就知道这一章的结论

### 第2层：重点指标表（2-6行，🔴🟢标记）
- 把最重要的指标放前面
- 每行：方向 | 指标名+数值 | 解释 | 判断
- 不超过6行

### 第3层：支撑数据（完整财务 or 完整指标，⚪为主）
- 补充参考数据
- 用code block或简化列表
- 不占主要注意力

### 示例（基本面章节）：

```
> 核心矛盾：Forward PE 9.6x（历史均值27x，折让65%）= 极度低估。但营收-1.65%、EPS增速-8.08%= COVID红利彻底消退。机构持仓67%+近季净增持+0.42%= 大资金博弈反转。5/5 Q1财报是唯一催化剂。

| 方向 | 指标 | 解释 | 判断 |
|:---:|------|------|:---:|
| 🟢 | **Forward PE 9.59x** | 前瞻1年PE，历史均值27x，折让65% | 极度低估，核心买点 |
| 🟢 | **毛利率 66.23%** | 大型药企>60%，属高壁垒 | 护城河强 |
| 🟢 | **机构持仓 67% +0.42%净买** | 大资金博弈反转，近季加仓 | 机构看多 |
| 🔴 | **营收增速 -1.65%** | COVID疫苗/药物断崖 | 核心困境来源 |
| 🔴 | **EPS 5年预期 -8.99%/年** | 分析师极度悲观 | 长期增长无故事 |
| ⚪ | **Q1财报日 5/5** | EPS预期$0.72 vs Q4 $0.66(a) | 唯一催化剂 |

完整财务（finviz，USD）：
PE(TTM)=19.9x | EPS=$1.36 | 营业利润率=27.99% | 净利润率=12.38%
ROE=8.87% | ROA=3.67% | ROIC=5.15% | 负债/权益=0.78
```

## 三、技术面三层展示规范

### 第1层：近期每日涨势 + K线要点

#### ① 近30日每日收盘价表格（新增，v5.1）
```
| 日期 | 收盘价 | 涨跌% | 成交量(万手) | 备注 |
|------|--------|-------|-------------|------|
| 2026-05-07 | $79.17 | +8.53% | 17.5M | 放量突破 ✅ |
| 2026-05-06 | $72.95 | -0.32% | 12.1M | 缩量整理 |
| ... | ... | ... | ... | ... |
```
- **格式**：日期 | 收盘价 | 涨跌%（🔴正=涨，🟢负=跌）| 成交量 | 备注（关键事件标注）
- **来源**：Yahoo Finance Chart API 1y（251条日线）— 本地代理7897
- **用途**：一眼看清近期走势结构，是单边上涨/下跌/震荡，帮助判断当前位置
- **⚠️ 注意**：
  - 港股数据：港股无完整逐笔/日线历史，改为标注近5日关键事件
  - A股数据：支持东方财富日线数据，同上格式
  - 若数据不可得（如API超时），标注`⚪数据暂缺`，不填假值

#### ② 关键日期要点（3-5个标志性日期）
- 选取对趋势有指示意义的日期（如突破日、放量日、消息日）
- 格式：`日期 🔴/🟢 +/-涨跌幅 成交量备注 — 一句话解读`
- 附关键价位表（阻力 → 支撑）

### 第2层：核心指标（6行内，🔴🟢标记）
- 均线排列 + MACD状态 + RSI + 布林位置 + KDJ
- 每行：方向 | 指标 | 解释 | 判断
- 不超过6行

### 第3层：完整指标（补充参考，⚪标记）
- ATR + 量比 + 52周分位 + YTD等
- 用code block简列

### MACD标注规则（重要！）
- DIF≈DEA 且 HIST=0 → **⚪临界**（不是死叉也不是金叉）
- DIF>DEA 且 HIST>0 → 🔴金叉
- DIF<DEA 且 HIST<0 → 🟢死叉

### 布林带标注规则
- position ≥ 0.90 → "上轨之上(超买)"
- position ≈ 0.85 → "上轨附近"
- position ≈ 0.50 → "中轨附近"
- position ≤ 0.15 → "下轨附近(超卖)"

## 四、L4风控裁决双表规范（v5.0新增）

### 表1：否决检查表（含「解释」列）

| 否决检查 | 结果 | 解释 |
|---------|:---:|------|
| 技术面否决（均线空头+MACD死叉） | ⚠️ 触发但不一票否决 | RSI 39.5未达超卖区(<30)，若RSI<30则触发；当前属警告但不阻断 |
| 资金面否决（主力净流出3日） | ✅ 不触发 | MFI(5/14/20日)全部>50，不否决 |
| 基本面否决（双恶化） | ✅ 不触发 | 营收-1.65%未达<-10%否决阈值 |
| 情绪面否决（极度乐观>0.85） | ✅ 不触发 | 无乐观信号 |

### 表2：维度权重检查表（含「说明」列）

| 维度 | 权重 | 调整 | 说明 |
|------|:---:|:---:|------|
| 技术面否决项：-0.15 | 15% | -15%×1 | 均线空头排列+MACD死叉双重压制，×1 |
| 基本面否决项：0 | 20% | 0 | 营收-1.65%接近但未达否决线 |
| 资金面否决项：0 | 25% | 0 | MFI全流入，不否决 |
| 情绪面否决项：0 | 15% | 0 | 无乐观泡沫信号 |

### 裁判综合评分 + 裁决

```
基础分：0.50
否决项扣减：-0.15（技术面）
裁判综合评分 = 0.35

最终裁决：WATCH（BUY≥0.55 / WATCH≥0.35 / else REJECT）

裁决理由：0.35落在WATCH区间但不REJECT，原因是Forward PE极低(9.6x)
构成底部保护，机构持仓67%说明大资金未离场，给出WATCH而非REJECT。
```

## 六、L3量化轨格式

每维度格式：
```
维度名称: 得分 × 权重 = 加权分 | 计算逻辑 | 指标解释
```

示例：
```
基本面 55分 × 30% = 16.5分
  | Forward PE低估撑起(+30) + 毛利率健康(+15) + 机构持仓(+5)
    - ROE偏弱(-10) - 营收负增长(-15) - EPS5年预期极低(-10)
  | Forward PE是核心买点撑起评分，负增长压制评分，整体困境反转特征明确
```
